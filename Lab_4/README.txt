Диапазон данных:
X<=2.934,
X>=101.89

Программа состоит из двух классов Main и Messages. 

В свою очередь Messages состоит из 6 методов:
1. inputNumber() - отвечает за ввод данных и проверку на корректность этих данных;
2. checkUserNumber() - отвечает за проверку данных введеных пользователем, а именно входят ли эти данные в заданный диапазон;
3. printInvalidTypeExceptionText() - отвечает за вывод сообщения об неправильно введенным данным;
4. printInputText() - отвечает за вывод сообщения о вводе числа;
5. printInvalidRangeExceptionText() - отвечает за вывод сообщения об неправильно введенном диапазоне данных;
6. printResult() - отвечает за вывод результата.

Main состоит из 6 методов:
1. main() - отвечает за вызов всех методов и классов;
2. calculateEquationOne() - отвечает за подсчет результата по формуле один;
3. calculateEquationTwo() - отвечает за подсчет результата по формуле два;
4. calculateEquationThree() - отвечает за подсчет результата по формуле три;
5. calculateEquationFour () - отвечает за подсчет результата по формуле четыри;
6. pow() - отвечает за поднесение числа в степень.
